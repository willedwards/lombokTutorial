# lombokTutorial

## Pre-requisites

 - Install lombok intellij plugin

 - Add the following dependency to your pom.xml
 
 ```
     <dependency>
      <groupId>org.projectlombok</groupId>
      <artifactId>lombok</artifactId>
      <version>1.16.18</version>
      <scope>provided</scope>
    </dependency>
 ```
